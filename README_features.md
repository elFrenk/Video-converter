# GUI Enhanced — Funzioni principali

## Cosa aggiunge rispetto alla GUI base

La versione enhanced introduce:

* preview del frame selezionato con slider
* validazione dei campi
* preset rapidi
* memoria delle ultime impostazioni
* test rapido
* popup risultati più leggibili
* separazione visuale in sezioni
* riepilogo export previsto

## Sezioni della GUI

La finestra è organizzata in blocchi:

* **Video e output**
* **Export base**
* **Coppie di frame**
* **Azioni**
* **Anteprima**
* **Riepilogo export previsto**
* **Stato**

## Preview

La preview permette di:

* scegliere il frame con uno slider
* saltare direttamente a:

  * primo frame
  * frame iniziale
  * frame finale
* verificare rapidamente che il video sia quello giusto

## Validazione

La GUI controlla i campi prima dell’export e segnala errori locali nelle sezioni interessate.

Esempi:

* file video inesistente
* output mancante
* frame range non valido
* resize compilato a metà
* qualità/compressione fuori range
* parametri pairs non validi

## Riepilogo previsto

Prima dell’export, la GUI stima:

* numero frame previsti
* numero coppie previste
* numero file attesi
* modalità attiva
* cartella output
* naming selezionato
