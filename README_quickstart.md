# Quickstart

## Requisiti

* Python 3.10 o superiore consigliato
* `opencv-python`
* `Pillow`
* `tkinter`

Installazione tipica:

```bash
pip install -r requirements.txt
```

## Avvio

Per la GUI avanzata:

```bash
python video_to_frames_gui_enhanced.py
```

## Workflow consigliato per geopyv

1. seleziona il video
2. leggi i metadati
3. controlla la preview con lo slider
4. applica il preset **geopyv standard**
5. scegli la cartella output
6. fai un **test rapido**
7. controlla i file in `quick_test`
8. lancia l’export completo
